# cc-marketplace

shnaki の個人用 Claude Code マーケットプレイス。

## インストール

```
/plugin marketplace add shnaki/cc-marketplace
/plugin install dev-tools@shnaki
```

## スキル一覧

### プロジェクト初期化

| スキル | 説明 |
|--------|------|
| `/init-python-project` | Python プロジェクトの CLAUDE.md を生成 |

### メタスキル（マーケットプレイス管理）

| スキル | 説明 |
|--------|------|
| `/bump-version` | plugin.json のバージョンをインクリメント |
| `/create-skill` | 新スキルのテンプレートを生成 |
